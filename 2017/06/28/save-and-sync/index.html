Emacsでバッファ保存時にブラウザをリロードする。 2017/06/28

最近、エディタをatomからEmacsに移行していてatomではbrowser-refreshというパッケージがありvagrantでWeb開発している時に便利だったのでEmacsで似たようなのが無いか調べた所、applescriptで実現可能そうなのでやってみた。

applescriptでchromeのアクティブなページを更新するスクリプトの準備

まずはスクリプトエディタを開きreload.scptという名前で~/.emacs.d/script/に保存する。

tell application "Google Chrome" to reload active tab of window 1

次にinit.elに下記を記載

;;chrome更新save-rehash
(defun reload-chrome ()"Save and reload browser"
(interactive)
(save-buffer)
(shell-command "osascript ~/.emacs.d/script/reload.scpt")
)
(global-set-key (kbd "<f5>") 'reload-chrome)

これでF5を押すことによってバッファの保存とchrome更新が同時にできるので効率UP。
