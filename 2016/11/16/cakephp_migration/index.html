Cakephp2.8 migrationの導入 2016/11/16

設計書管理システムの刷新のためデモ用のシステムを構築するので
今後、テーブル定義が変更しても良いようにMigrationを入れる。

CakePHP3.x~はデフォルトでmigrationsが入っているらしい

  • Cookbook

手順としては

  1. Migrationをgitコマンドで入れる。
  2. bootstrap.phpでMigrationsを読み込む
  3. 初期マイグレーションの作成
  4. 適宜マイグレーションファイルを作成する

プロジェクトがあるフォルダで行う

git submodule add \
git://github.com/CakeDC/migrations.git app/Plugin/Migrations

app/Config/bootstrap.phpに記載する

CakePlugin::load(array('Migrations'));

初期マイグレーションを作成する。

Console/cake Migrations.migration run all -p

適宜スキーマを変更した際に行う。

Console/cake Migrations.migration generate -f

以上。詳細はまた、おいおい更新していく予定です。
