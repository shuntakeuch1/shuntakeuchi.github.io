数値型

よく使うのはint・float・boolかな。
桁数で困るほどそんなに大きい数値を使わない事が多い。(42億って。)

Type ビット数 補足
bit 1-64 デフォルトは0
tinyint 0-255 符号付きの範囲は-128~127
bool 0-255 0はfalse,0以外はture
int 0-4294967295 -2147483648 - 2147483647
bigint 0-18446744073709551615 -9223372036854775808 - 9223372036854775807
     

それ以外もsmallintやmidumintなどがあるがintより小さいサイズであり、固定小数点のdecimalや浮動小数点のfloat、doubleなどを選ぶ事ができる。

個人的にはbool(boolean)が内部的にはtinyintの動きをしているのが衝撃だった。
(疑っていたが実際に調査している人がいて分かりやすかった。)

文字列型

Type 長さ 補足  
char 0-255 固定長文字列、長さを省略すると長さは1になる  
varchar 0-65535 可変長文字列、UTF8では最大21,844 文字  
tinytext 255 長さは指定不可である  
text 1-65535    
mediumtext 1-16777215    
longtext 1- 4,294,967,295    

ほかにも画像などのバイナリデータを入れる属性としてcharであればbynary、
varcharならばvarbynary、textのblobなどがシノニムとなっている。
